# hacker-news-scraper-json
Python web scraper that extracts Hacker News posts including title, author, score, and comments, and saves them to JSON.
# Hacker News Scraper (JSON)

This project is a Python web scraper that extracts data from Hacker News.

## Features

- Scrapes titles
- Extracts authors
- Extracts scores
- Extracts comment counts
- Handles pagination
- Saves data to JSON file

## Technologies used

- Python
- requests
- BeautifulSoup
- JSON

## Output example

```json
{
  "title": "Example title",
  "author": "username",
  "score": "100 points",
  "comments": 42
}
